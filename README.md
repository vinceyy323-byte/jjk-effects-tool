# ⚡ JJK Effects Tool - Interactive Jujutsu Kaisen Animation

An interactive web-based tool for creating stunning Jujutsu Kaisen anime effects with smooth animations, real-time controls, and customizable parameters.

## 🎮 Features

### Animation Modes
- **Gojo Blue** - Smooth blue cursed energy sphere with particle effects
- **Gojo Red** - Explosive red cursed energy with expanding waves
- **Purple** - Converging purple energy with multiple ripple rings
- **Sukuna** - Chaotic white cursed energy with rotating lines
- **Mahoraga** - Adaptive golden wheel with rotating spokes
- **Blackflash** - Screen-filling flash impact with black cross
- **Fuga** - Orange tornado spiral with particle trail
- **Todo** - Yellow clap energy burst effect

### Interactive Controls
- **Mouse Click** - Spawn effects at cursor position
- **Keyboard Shortcuts** - Quick mode switching (Y, P, S, M, B, F, T)
- **Real-time Sliders** - Adjust particle size, animation speed, and intensity
- **Visual Effects Toggle** - Enable/disable glow, trail, and pulse effects
- **Live FPS Counter** - Monitor performance

### UI Features
- Modern glassmorphism design
- Responsive control panel
- Smooth animations and transitions
- Real-time effect customization
- Keyboard shortcuts guide

## 🚀 Quick Start

1. **Open in browser**: Simply download/clone and open `index.html` in a web browser
2. **Spawn Effects**: Click anywhere on the canvas or use keyboard shortcuts
3. **Customize**: Adjust sliders and toggle effects in the control panel
4. **Switch Modes**: Click mode buttons or press keyboard keys

## ⌨️ Keyboard Shortcuts

## 🎛️ Controls

### Animation Settings
- **Particle Size** (5-50px) - Controls the size of effect elements
- **Animation Speed** (0.5x-3x) - Multiplier for animation speed
- **Intensity** (0.3-1.0) - Glow and shadow intensity

### Visual Effects
- **Glow Effect** - Enable/disable shadow glow
- **Trail Effect** - Enable/disable fade trail on canvas
- **Pulse Effect** - Enable/disable pulsing animation (future enhancement)
